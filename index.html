The task presented as a challenge by "VEGA IT" company.

The goal of the task is to create an algorithm that calculates the fields that are not flooded, represented by white squares. The user will mark the fields on the grid that they want to represent a road, indicated by black squares, and the algorithm should color the squares in blue, indicating the squares that have a "road" (black square) on both their left and right sides.

Zadatak koji je predstavljen kao izazov od strane firme "VEGA IT".

Cilj zadatka je da se napravi algoritam koji će izračunavati polja koja nisu poplavljena, bele kvadratiće. Korisnik će na matrici obeležavati polja koja želi da predsavljaju put, odnosno crne kvadratiće, a algoritam treba da oboji kvadratiće u plavo, odnosno one kvadratiće koje i levo i desno od sebe imaju "put" (crni kvadratić).

A website that will make it easier for me to track my collection.

I have a collection of cans (mostly beer cans) that is becoming increasingly difficult to keep track of due to its growing size. The idea of this website is to easily check whether I have a particular can or not.

There will be an SQL server that will contain a table of cans with all the associated parameters, including an image of the can itself. By reading data from the SQL, I will create a web page that will display all this information. There will also be an option to add a new can to the SQL table via the website.

Web sajt koji će mi olakšati praćenje kolekcije.

Posedujem kolekciju limenki (pretežno limenki piva) koju je, sa sve većim obimom, sve teže ispratiti. Ideja ovog sajta je da uvek lako mogu proverit da li posedujem određenu limenku ili ne.

Postojaće SQL server, koji će sadržati tabelu limenki sa svim pratećim parametrima kao i sa slikom same limenke. Čitajući podatke iz SQL-a, napraviću web stranicu koja će prikazivati sve te podatke. Takođe će bit opcija za ubacivanje nove limenke u SQL tabelu preko sajta.
